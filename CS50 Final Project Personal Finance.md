**CS50 Final Project:** *Personal Finance Tracker*



--



**Idea**: A web app where users can record their income and expenses, categorize them, and view summaries and charts of their spending.



---



**Tech Stack**



Backend: Python + Flask

Database: SQLite (built-in, CS50-friendly)

Frontend: HTML/CSS + optional JS for interactivity

Optional: Chart.js or a simple table for visualizing expenses



---



**Features**



1\. User Accounts

* &nbsp;	Register
* &nbsp;	logout
* &nbsp;	login

Each user sees only their own transactions after registration and log-in



2\. Add Transactions

* &nbsp;	Fields: amount, type (income/expense), category (food, rent, entertainmentâ€¦), date
* &nbsp;	Optional: notes field



3\. View Transactions

* &nbsp;	Table view sorted by date
* &nbsp;	Filter by category or date range



4\. Summaries / Analytics



* &nbsp;	Monthly income vs. expenses
* &nbsp;	Total spent per category
* &nbsp;	simple bar/pie chart
* &nbsp;	CSV export/import for transactions



---



**Implementation Roadmap**



Step 1: Set up Flask app and SQLite database

Step 2: Create user registration/login (if doing accounts)

Step 3: Build forms to add transactions

Step 4: Display transactions in a table

Step 5: Add filtering and summary calculations

Step 6: Add charts or graphs for visual summary
